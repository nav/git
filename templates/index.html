Hello, my name is Navdeep Aulakh but I usually go by Nav. I am a Web Applications Developer from Vancouver, BC and I make cool web applications using Python and Django. To add a little spice to the Django apps, I often use jQuery along with Backbone.js. I have worked with PHP and Java prior to moving on to Python. My favourite desktop operating system is Mac OS X and I love running Debian as a server OS.
